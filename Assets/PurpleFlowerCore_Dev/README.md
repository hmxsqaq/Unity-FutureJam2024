# PurpleFlowerCore使用指南

> 当前版本:0.3.1

## 简介

Unity游戏开发中，程序框架的使用已经非常广泛，在一段时间的学习和使用之后，我不甘于使用别人写好的程序框架，遂抱着学习、锻炼和更好的个性化的目的制作自己的程序框架，即PurpleFlowerCore(简称PFC)。

其中大部分是造轮子，小部分是我自创的功能。才疏学浅，漏洞百出，请多指教。
